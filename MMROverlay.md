# 📐 MMR Overlay Setup

## 1. Configure the Scoreboard Key

First, register the key you use to display the scoreboard in Rocket League.

- Open RLVision
- In the **Scoreboard Key** section, click **Bind**
- Press the keyboard key or controller button you use for the scoreboard in RL
- The key is saved automatically

---

## 2. Position the Overlays

Positions must be configured **once per game mode** (1v1, 2v2, 3v3).

### Step 1 — Launch a private match

Launch a private match in the mode you want to configure:
- Private match **1v1** → configures 1v1 mode
- Private match **2v2** → configures 2v2 mode
- Private match **3v3** → configures 3v3 mode

### Step 2 — Move the overlays

- Hold your **scoreboard key** (keyboard) to display the RL scoreboard
- Hold **Alt** to activate the overlay drag mode
- Drag each overlay to the desired position
- Click **Save** to save the positions

### Step 3 — Overlay order ⚠️

Each overlay is already assigned in order — you just need to place them in the right spot:

- **1st overlay** → 1st player in the scoreboard
- **2nd overlay** → 2nd player in the scoreboard
- And so on...

Place them **from top to bottom**, in the same order as the RL scoreboard:

```
↑  1st overlay → 1st player in scoreboard
   2nd overlay → 2nd player in scoreboard
↓  3rd overlay → 3rd player in scoreboard
```

This applies separately for each team (blue and orange).

### Step 4 — Repeat for each mode

Repeat steps 1 to 3 for all three modes. Positions are saved independently for each mode.
